# Overview

Purpose:
- Frontend for a conversational Weather AI: users authenticate, start conversations, and ask weather-related questions.

Key goals:
- Fast developer feedback loop with Vite.
- Clear separation of UI, business logic (composables), and API services.
- Simple authentication and secure token handling.

Users:
- End users seeking weather info through conversational UI.
- Developers extending chat features or integrating additional data sources.
