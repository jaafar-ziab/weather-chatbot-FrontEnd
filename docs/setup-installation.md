# Setup & Installation

Prerequisites:
- Node.js 16+ installed
- npm (or yarn)

Local development:

```bash
npm install
npm run dev
```

Environment variables (Vite):
- Create `.env` or `.env.local` with `VITE_API_BASE_URL` set to your API.

Production build:

```bash
npm run build
npm run preview
```

Helpful tips:
- Use Volar in VSCode for Vue 3 support.
- Use the browser devtools and Vite overlay for quick debugging.
